How to Join the Server

Step-by-Step Guide to Join Eric's Minecraft Server

  1. Contact the admin: The admin should give you the Zerotier network ID and the server IP. If you don't know the admin, you are probably not allowed to join the server.
  2. Install TLauncher: Download and install TLauncher, the best Minecraft launcher: https://tlauncher.org/
  3. Install ZeroTier: Install ZeroTier Network to be able to connect to our private network: https://www.zerotier.com/download
  4. Join ZeroTier network: Assuming the admin has provided you with the network ID, follow these steps:
    • For Windows:
      1. Open Command Prompt.
      2. Navigate to the ZeroTier installation directory by running: cd "C:\Program Files (x86)\ZeroTier\One"
      3. Join the network using: zerotier-cli join <network_id>
    • For Linux:
      1. Open a terminal.
      2. Run the following command to join the network: sudo zerotier-cli join <network_id>
    • For macOS:
      1. Open a terminal.
      2. Run the following command to join the network: zerotier-cli join <network_id>
  5. Wait for the admin: The admin should accept you in the private network. Contact him for confirmation.
  6. Install version Forge 1.19.2: From the TLauncher interface, select the version Forge 1.19.2 and install it. IMPORTANT! No Optifine, some mods don't work with Optifine, other alternative mods will be installed to optimize the game.
  7. Download the Mods: Click the button below to download the required mods folder. Extract the contents and place them into your TLauncher `mods` directory. You can find it by opening TLauncher application, clicking on the folder icon on the bottom right corner, and searching for the 'mods' directory. Place there your .jar files.
    Download Mods Folder
  8. Join the server: Launch Minecraft, go to "Multiplayer" and add the server using the IP address provided by the admin.
  9. Start Your Adventure: Connect and start exploring the amazing world we have created!
